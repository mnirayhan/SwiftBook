## Material v3 Design Refresh & Core Stability Improvements

SwiftBook v1.6.0 introduces a refined Material v3 interface, improved layout behaviour, and multiple enhancements aimed at stability, performance, and a more consistent user experience.

---

## Material v3 Enhancements
- Updated interface to a more polished and cohesive Material v3 design.
- Improved visual consistency, spacing, and component alignment.
- Expanded translation support to ensure broader international usability.

---

## Layout and Behaviour Updates
- **Breaking Change:** Messenger support has been removed from the default mobile layout.
- Users may access Messenger by switching to **Desktop Mode**.
- Corrected an issue where orientation changes caused unintended page resets.
- General interface refinements for smoother navigation and improved clarity.

---

## Reel Hiding Improvements
- More reliable reel detection across layout variations.
- Stabilized behaviour on both mobile and desktop environments.
- Improved compatibility with newer Facebook interface updates.

---

## Codebase and Resource Optimization
- Streamlined internal code for better maintainability.
- Reduced redundant resources to ensure cleaner build structure.
- Improved runtime performance and overall app responsiveness.

---

## Bug Fixes
- Increased reliability of reel hiding on various Facebook layouts.
- Minor interface and behaviour corrections to support a smoother user experience.

---

> **Note:**
> Messenger access is available only in Desktop Mode.
> This update focuses on improved design consistency, stability, and long-term performance.
